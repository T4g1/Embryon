Detailed Description

Gére l'affichage et les différents rendu via Irrlicht.

Tout les appels concerant Irrlicht sont confinés dans ce Module, la création de chaque menu se fait par l'une des méthodes de ce Module.

La création du rendu des différents éléments du jeu se fait dans ce Module. Gére également les collisions du joueur avec l'environnement.
